Estimated: deletes ~600 LOC across 8 files. Highest single-source-of-truth gain.

2. **Pick canonical token names** (`--wp-bg-2/--wp-bg-3/--wp-text-muted/--wp-text-dim`) and remove all `bg2/bg3/text2/text3` usages with a codemod. The aliases at tokens.css:2239-2245 can stay for backward compat, but Vue files should consistently use canonical names. Particularly impacts `PipelineSteps`, `PipelineStepPicker`, `HistoryPanel`, `DerivationRuleCard`.

3. **Standardize Button icon prop** to a single form (recommend bare `pi-foo` since `Icon.vue:12` already strips the prefix — and codify in the Button JSDoc). Run a one-line codemod to convert all `icon="pi pi-foo"` → `icon="pi-foo"`.

4. **Consolidate empty-state**: make `wp-empty` the canonical pattern; rewrite `wp-comm-empty` as a thin alias. Add a shared `wp-empty-card` for dashed-border empty cards (DerivationEditor, ConstraintEditor, PipelineSteps).

5. **Extract `RichTextInput`/`RichTextPreview` shared syntax-highlight CSS** to `styles/rich-text.css` and remove the literal `#34d399`/`#fb923c` hexes (replace with new `--wp-rt-token-good`/`--wp-rt-token-mut` tokens).
